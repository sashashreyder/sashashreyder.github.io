HTML Cheatsheet

Tags and Descriptions

Tag Name Description
<div> Division Contaner that divides the page into sectors.
<br> Break Vertical line breaker.
<nav> Navigation Can be used inside of the header or on its own.
<article> Article Holds content that make sense on its own: blogs. magazines etc
<figure> Figure Encapsulate media such as images, illustration, diagram etc.

Attributes

Tag Name Description
href Hypertext Reference Specifies the URL of the linked resource. This attribute is commonly used with anchor <a> elements to define the destination of the hyperlink.
src Source Specifies the URL of the image or other media file to be embedded. It is commonly used with <img>, <video>, and <audio> elements to define the source of the media.
alt Alternative Text Provides alternative text for an image if the image cannot be displayed. It is used with the <img> element to improve accessibility for users who may be using screen readers or have images disabled in their browser.
class Ð¡lass Specifies one or more class names for an element, allowing you to apply CSS styles or JavaScript behavior to multiple elements with the same class. Classes are commonly used for styling and organizing content.
type Type Specifies the MIME type of the content linked to by the <a> element when the href attribute is a mailto: link or points to a resource that requires a specific MIME type. It is also used with input elements such as <input> and <button> to specify the type of control.
